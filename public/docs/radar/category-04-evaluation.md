## 一、技术概述

**Skill 评测**是贯穿 Skill 全生命周期的量化评估与质量定级体系，核心目标是构建可量化、可复现的多维度指标，以客观数据衡量 Skill 好不好用、稳不稳定、安不安全、有没有价值。

## 三、代表技术与项目

### 3.1 Skill-insight

openEuler 社区推出的多维评测与执行追溯平台。

**核心能力**：
- 覆盖效果、效率、成本多维指标
- 实时生成执行流程图
- 智能归因分析

**链接**：[AtomGit](https://atomgit.com/openeuler/witty-skill-insight)

**详细页面**：[Skill-insight 详解](./tech-skillinsight)

### 3.2 SkillsBench

衡量智能体技能在多样化任务中表现的基准测试。

**核心发现**：
- 人工精选的 Agent 技能能显著提升任务表现（平均 +16.2%）
- 模型无法可靠生成对自身有效的技能
- 技能设计应遵循"少即是多"原则（2-3 个技能效果最佳）

**详细页面**：[SkillsBench 详解](./tech-skillsbench)

### 3.3 SkillProbe

用"魔法"打败"魔法"——用 Skill 审计 Skills。

**核心能力**：
- 安全漏洞检测
- 权限滥用检测
- 恶意行为识别

**详细页面**：[SkillProbe 详解](./tech-skillprobe)

### 3.4 agent-skills-eval

Agent Skills 测试运行器，支持 with_skill vs without_skill 对比评测。

**核心能力**：
- 同一 prompt 双向运行（with_skill / without_skill）
- Judge 模型评分，输出 pass/fail + 证据
- 静态 HTML 报告，无需基础设施
- OpenAI-compatible API 支持
- TypeScript SDK + CLI 双模式

**链接**：[GitHub](https://github.com/darkrishabh/agent-skills-eval)

### 3.5 SkillAudit

以技能为中心的评估审计框架，基线对比剥离backbone影响。

**核心能力**：
- 任意技能即输即评，自动生成定制审计
- PRG基线剥离归因，骨架强度不再污染技能效用
- 静态+动态双层安全检测
- 浏览器扩展嵌入发现页面

**详细页面**：[SkillAudit 详解](./tech-skillaudit-assessment)

### 3.6 Skill Coverage

借鉴代码覆盖率度量技能测试充分性。

**核心能力**：
- SBC可测试行为约束作为分母
- 双门覆盖判定（applicable + evidence）
- 人类一致率93.71%验证评判协议
- 任务成功不意味着技能被充分测过

**详细页面**：[Skill Coverage 详解](./tech-skill-coverage)

### 3.7 Tessl Skill Evaluation

配对对比量化技能边际贡献，双维度评分。

**核心能力**：
- 配对对比剥离技能边际贡献
- 完成度+遵循度双维度评估
- 单技能精确诊断
- 技能+便宜模型逼近旗舰模型，成本1/3以下

**详细页面**：[Tessl Skill Evaluation 详解](./tech-tessl-skill-evaluation)
